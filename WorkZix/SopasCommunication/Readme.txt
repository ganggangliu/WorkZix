SopasCommunication is a simple framework to demonstrate the communication with
SICK devices using the SOPAS communication architecture. The framework offers 
basic functionality to invoke device methods, subscribe to device events, read 
variables from the device and write variables to the device.

The full documentation is located in the subfolder "SopasCommunicationLms100\doc".
